🎯 How to Use NH Canvas

  1. Enable Design Mode using the button above
  2. Click any element to select it (you'll see a blue outline)
  3. Drag elements around to reposition them
  4. Resize elements using the corner handles
  5. Multi-select by holding Shift and clicking multiple elements
  6. Use alignment tools from the context toolbar
  7. Your changes persist across page refreshes!
